Class TimeControlModePatch

Code patcher through Harmony which overrides new campaign map time speed on its setter.

More about it on issue #108 and issue #133

Check SetTimeSpeedPatch too, it uses the flag CanSyncTimeControlMode

Remarks

This class prefixes setter of TaleWorlds.CampaignSystem.Campaign.TimeControlMode to change its behavior and change its value just after TaleWorlds.CampaignSystem.Campaign.SetTimeSpeed(System.Int32) is called
