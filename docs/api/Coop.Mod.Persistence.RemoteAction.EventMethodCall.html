Class EventMethodCall

RailEvent used to initiate remote procedure calls. The way this event is processed differs between client and server:

  • The server acts as a broadcasting relay station. Effectively receiving the event and forwarding it to all clients (INCLUDING the client that sent the event!). Note that the server will delay event execution until all arguments have been transferred to all clients.
  • Clients will resolve the call and arguments locally and execute it.
