Invokable

Type erased invocation wrapper for patched methods. Every invokable instance is registered with Registry on every client.

PatchedInvokable

Type erased invocation wrapper for patched methods. On creation, a snapshot of the IL representation of the method is stored internally. The snapshot includes all patches to the function already applied. The snapshot can be called using InvokeOriginal(Object, Object[]).
