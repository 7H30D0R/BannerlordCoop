Classes

ActionBehaviourBuilder

Builder class to define the behaviour of a patched method call, property getter / setter or monitored field change.

ActionValidator

Static lookup table to check if an action may be executed right now. Validators are created trough FieldBehaviourBuilder and CallBehaviourBuilder.

ActionValidatorRegistry

Registry for action validators. For internal use only, public access through ActionValidator.

CallBehaviourBuilder

Builder class to define the behaviour of a patched method or property getter / setter.

CamBeNullAttribute

Condition

A boolean operator that evaluates the EOriginator and an System.Object instance.

ConditionalBehaviour

Describes a behaviour that only applies when a condition is met.

FieldAccessBehaviourBuilder

Builder class to define the accessors to a field in order to define it's behaviour. Note that field changes cannot be intercepted (patched) directly, instead methods and property setters that access the field are patched - these are referred to as 'accessors' to the field.

FieldBehaviourBuilder

Builder class to define the behaviour of a managed field.

Util

Utility functions to work with Sync.Behaviour.

Interfaces

IActionValidator

Validator that checks if an action, i.e. a method call or field change, is valid or not.

IPendingMethodCall

Interface to process a pending method call.

ISynchronization

Interface to synchronize method calls and field changes to other clients.

Enums

ECallPropagation

Describes the behaviour of a patch after a prefix function has been executed.

EFieldChangeAction

Defines the action that should be taken when a monitored field was changed by a known accessor to the field.

EOriginator

Defines the origin of an action, i.e. who made a method call, property getter / setter call or a field change.
